# PDT Command Hub — Releases & Live Feed

Public distribution channel for the [PDT Command Hub](https://thepurposedriventrader.com/hub) desktop app by The Purpose Driven Trader.

- **Releases** — download the latest installer from the Releases tab. The app auto-updates from here.
- **`announce.json`** — the live in-app announcement banner (promos, discount codes, new videos). Updated automatically by the daily promo watcher.
- **`promo-sources.json`** — which prop-firm pages the watcher checks and the rules it follows.
